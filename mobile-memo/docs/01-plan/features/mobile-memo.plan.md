> **요약**: 모바일 브라우저에서 설치·실행 가능한 PWA 메모장 앱 (HTML/CSS/JS + Service Worker)
>
> **프로젝트**: mobile-memo
> **버전**: 1.0.0
> **작성자**: 사용자
> **날짜**: 2026-03-04
> **상태**: Draft

---

## 1. 개요

### 1.1 목적

스마트폰 브라우저에서 홈 화면에 설치해 앱처럼 사용할 수 있는 메모장을 만든다.
별도 앱스토어 배포 없이, 링크만 있으면 누구나 설치·사용 가능하다.

### 1.2 배경

- 빌드 도구·Node.js 설치 없이 HTML/CSS/JS만으로 모바일 앱 경험 제공
- LocalStorage로 오프라인 저장 — 인터넷이 없어도 메모 작성 가능
- PWA(Progressive Web App) 기술 학습 목적도 겸함

### 1.3 관련 문서

- 기술 스택: `CLAUDE.md`
- 구현 파일: `index.html`, `css/style.css`, `js/app.js`, `manifest.json`, `sw.js`

## 5. 위험 요소 및 대응

| 위험 | 영향도 | 발생 가능성 | 대응 방안 |
|------|--------|-------------|-----------|
| Service Worker는 HTTPS 또는 localhost에서만 동작 | Medium | High | Vercel/GitHub Pages로 배포하거나 localhost로 테스트 안내 |
| iOS Safari PWA 기능 제한 (push 알림 등) | Low | High | 핵심 기능만 구현, 고급 API 미사용 |
| LocalStorage 저장 한도 (5MB) | Low | Low | 메모 수가 많아질 경우 안내 메시지 추가 |

## 6. 아키텍처 결정

### 6.1 프로젝트 레벨

| 레벨 | 선택 | 이유 |
|------|:----:|------|
| **Starter** | ✅ | 빌드 없음, Vanilla JS, 초보자 친화적 |
| Dynamic | ☐ | |
| Enterprise | ☐ | |

### 6.2 핵심 기술 결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 언어 | Vanilla JavaScript | 의존성 없음, 즉시 실행 |
| 스타일 | 순수 CSS (모바일 퍼스트) | 빌드 불필요 |
| 데이터 | LocalStorage + JSON | 서버 없이 간단한 영속성 |
| 화면 전환 | JS로 섹션 show/hide | SPA처럼 동작, URL 변경 없음 |
| PWA | manifest.json + Service Worker | 설치 가능 + 오프라인 지원 |

### 6.3 화면 구성 (View)

```
┌────────────────────────────┐
│  [검색창]         [⋮ 메뉴]  │   ← 헤더
├────────────────────────────┤
│  메모 카드 1               │
│  제목 / 날짜 / 본문 미리보기│
├────────────────────────────┤
│  메모 카드 2               │
│  ...                       │
├────────────────────────────┤
│                     [＋]   │   ← 플로팅 버튼
└────────────────────────────┘

[＋] 클릭 → 편집 화면
메모 카드 클릭 → 편집 화면 (기존 내용 로드)

편집 화면:
┌────────────────────────────┐
│  [← 뒤로]    [💾저장] [🗑삭제]│
├────────────────────────────┤
│  제목 입력                 │
├────────────────────────────┤
│  본문 입력                 │
│  (textarea, 자동 높이)     │
└────────────────────────────┘
```
