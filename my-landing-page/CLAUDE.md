# my-landing-page

## Project Level
Level: Starter

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure
```
my-landing-page/
├── index.html       # Main landing page
├── css/style.css    # All styles
├── js/main.js       # Smooth scroll + form handler
├── images/          # Place your images here
└── docs/            # PDCA documents
    ├── 01-plan/
    └── 02-design/
```

## How to Run
Open `index.html` in your browser — no build step needed.
