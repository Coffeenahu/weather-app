# my-todo

## Project Level
Level: Starter

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Features
- 할 일 추가 / 완료 토글 / 삭제
- 전체 / 진행중 / 완료 필터
- LocalStorage로 새로고침 후에도 데이터 유지
- 남은 항목 수 표시
- 완료 항목 일괄 삭제

## File Structure
```
my-todo/
├── index.html      # 메인 UI
├── css/style.css   # 스타일
├── js/main.js      # 상태 관리 + 렌더링 로직
└── docs/           # PDCA 문서
```

## How to Run
index.html을 브라우저에서 열면 바로 실행됩니다.
